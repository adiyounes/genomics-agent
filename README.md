# Genomics Research Agent

[![CI](https://github.com/adiyounes/genomics-agent/actions/workflows/ci.yml/badge.svg)](https://github.com/adiyounes/genomics-agent/actions/workflows/ci.yml)

An AI-powered agent that retrieves and summarises the latest biomedical research for a given gene and condition. Built as a portfolio project demonstrating ReAct-pattern agentic AI, API orchestration, and full-stack development.

## How it works

When a user clicks "Latest Research" on a genomic variant, the agent:

1. Searches PubMed (NCBI E-utilities API) for recent papers on the gene/condition pair
2. Fetches abstracts for the top 5 results
3. Summarises findings using the Anthropic Claude API
4. Grades relevance f results are off-topic, it refines the query and retries (ReAct loop)
5. Returns a 3-5 bullet point summary to the user

## Tech stack

| Layer | Technology |
|---|---|
| Backend | Python, FastAPI, uvicorn |
| AI | Anthropic Claude (claude-sonnet-4-20250514) |
| Biomedical data | NCBI PubMed E-utilities API |
| HTTP client | httpx (async) |
| Frontend | React + Vite |

## Setup

```bash
# 1. Clone the repo
git clone https://github.com/adiyounes/genomics-agent.git
cd genomics-agent

# 2. Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r backend/requirements.txt

# 4. Add your Anthropic API key
cp backend/.env.example backend/.env
# then edit .env and paste your key from console.anthropic.com

# 5. Run the server
cd backend
uvicorn main:app --reload
```

The API will be available at `http://localhost:8000`.  
Interactive docs (auto-generated by FastAPI) at `http://localhost:8000/docs`.

## API

```
GET  /health              Health check
POST /research            Run the research agent
```

**POST /research** request body:
```json
{
  "gene_name": "BRCA1",
  "condition_name": "breast cancer"
}
```
